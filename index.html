JavaScript Promises Examples

See your browser's DevTools console for output of each demo.

The Promise object

An example of writing a Promise, with handling of both 'resolve' and 'reject' outcomes.

async / await

These keywords provide alternative syntax to writing thenables. Use of async/await create promises that read like synchronous procedural code.

Asynchronous network example

Typical example of using asynchronous JavaScript to load data from a remote server.

promise.finally()

Returns a new promise that runs after all other promises have settled. This will always execute, much like the finally of a try...catch...finally block.

Promise.allSettled()

Given an array of promises, once all have settled, allSettled() collects the status and result of each promise and returns them in a new promise.

Promise.any()

Given an array of promises, any() will return a new promise with the result of the first fulfilled (resolved) promise. If all promises of the array reject, then an AggregateError object of reasons will be returned.

Promise.race()

Returns a new promise with the result of the first promise that's settled, whether it's resolved or rejected.
